The advantages

  • Independent of resolution
  • Supported in all modern browsers
  • Relevant in the future
  • Easy to create and edit
  • Modified by CSS & JS
The advantages of svg graphics are 1) SVG is a vector format, which is why it is worth using it for responsive websites, where the size of images depends on the resolution of the user’s display. 2) svg is compressible 3) SVG images are very easy to manage, and this opens up even more opportunities for designers who can change color, add shadow, filters, blur and many more other effects. 4) SVG is pretty easy to understand. 5) SVG works with open web standards

The advantages

  • Compressible
  • Text format
  • Scalability
  • Supported scripting languages based on the ECMAScript specification
  • SVG is an open standard
6) Text format - SVG files can be read and edited using ordinary text editors. When viewing documents containing SVG graphics, there is access to viewing the code of the file being viewed and the ability to save the entire document. In addition, SVG files are usually smaller in size than JPEG or GIF images of comparable quality and can also be compressed well. 7) Scalability - SVG is a vector format. It is possible to enlarge any part of the SVG image without losing quality. Additionally, it is possible to apply filters to the elements of an SVG document — special modifiers for creating effects similar to those used in processing bitmaps. 8) The use of raster graphics in SVG documents is widely available. It is possible to insert elements with images in PNG, GIF or JPG formats. 9) Animation is implemented in SVG using SMIL (Synchronized Multimedia Integration Language). Supported scripting languages based on the ECMAScript specification. SVG elements can be controlled using javascript. Using scripts and animations in SVG allows you to create dynamic and interactive graphics. SVG provides an event model, events are tracked. Animation can be run on a specific event, which makes the graphics interactive. Each element has its own events to which individual scripts can be attached.
